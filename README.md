# EduHelperBot

EduHelper — учебный Telegram-бот-помощник для школьников и студентов.

## Возможности

- Отвечает на вопросы по математике, физике, истории, биологии и другим предметам
- Объясняет темы простым языком и приводит примеры
- Генерирует тесты и проверяет ответы через inline-кнопки
- Делает краткое summary по тексту или ссылке
- Отслеживает прогресс пользователя в SQLite

## Установка

1. Перейдите в папку проекта:

```bash
cd /Users/jurasik/Downloads
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` рядом с `EduHelper.py` и добавьте токен Telegram:

```text
TELEGRAM_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_openai_ключ  # необязательно
GROQ_API_KEY=ваш_groq_ключ      # необязательно
```

## Запуск

```bash
python3 EduHelper.py
```

## Примечания

- Файл `.env` добавлен в `.gitignore`, чтобы секреты не попали в репозиторий.
- Если не задан `OPENAI_API_KEY` или `GROQ_API_KEY`, бот будет работать только с базовой логикой без AI-ответов.
- Для использования AI-ответов нужен действующий API-ключ OpenAI или Groq.
